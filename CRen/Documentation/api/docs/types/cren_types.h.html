Opaque Objects:

Context:

typedef struct CRenContext CRenContext;

An opaque object that manages most API resources. It's an implementation detail that the API handles automatically, users can't directly manipulate it, only through functions.

Camera:

typedef struct CRenCamera CRenCamera;

An opaque object that serves as a unique identifier for camera references. Its purpose is to provide a consistent handle that users can employ to specify which camera they are referring to in API calls.

Texture2D:

typedef struct CRenTexture2D CRenTexture2D;

An opaque object that serves as a unique identifier for 2D textures. Its purpose is to abstract the underlying data and provide controlled access exclusively through designated API functions.

Quad:

typedef struct CRenQuad CRenQuad;

An opaque object that serves as a unique identifier for quads. Its purpose is to abstract the underlying data and provide controlled access exclusively through designated API functions.
