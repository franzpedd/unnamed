BufferConstant

File: cren_buffer.h

Type: C Struct

Description

Stores camera data accessible across all shaders via a Uniform Buffer Object (UBO), offering optimal performance for scene-wide constants. Data is uploaded once per frame and reused globally. 

typedef struct BufferCamera
{
    align_as(16) fmat4 view;
    align_as(16) fmat4 viewInverse;
    align_as(16) fmat4 proj;
} BufferCamera;

Members

view:

The camera's view matrix.

viewInverse:

The camera's view inverse matrix, this is calculated on CPU and maintaned internally because is faster.

proj:

The camera's projection matrix, calculated on CPU as it's faster.
