BufferConstant

File: cren_buffer.h

Type: C Struct

Description

Stores quad data accessible via a Uniform Buffer Object (UBO), offering optimal performance for scene-wide constants. Data is uploaded once per quad. 

typedef struct BufferQuad
{
    align_as(4) float billboard;
    align_as(4) float uv_rotation;
    align_as(8) float2 lockAxis;
    align_as(8) float2 uv_offset;
    align_as(8) float2 uv_scale;
} BufferQuad;

Members

billboard:

If 1.0f enables the quad to act as a billboard (always facing the camera), acts like an in-place quad otherwise.

uv_rotation:

Rotation of the UV-Coordinates, for mapping with the 2D texture.

lockAxis:

When using billboad, allow the locking of axis but still acting like a billboard. (If both axis are locked it'll stay in-place)

uv_offset:

Allows to offset the UV-Coordinates in order to add freedom to adjust textures in-renderer.

uv_scale:

Allows to scale the UV-Coordinates in order to add freedom to adjust textures in-renderer.
