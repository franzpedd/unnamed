BufferConstant

File: cren_buffer.h

Type: C Struct

Description

Holds per-frame object data transmitted to shaders via PushConstants, which enables extremely fast GPU updates. The buffer is limited to 128 bytes in total, with 80 bytes(64 + padding) currently allocated. 

typedef struct BufferConstant
{
    align_as(4) uint32_t id;
    align_as(16) fmat4 model;
} BufferConstant;

Members

id:

The identification number to track the object this buffer associates with, usually a renderable.

model:

The object's model matrix, the transformation matrix that defines where and how a 3D model exists in the world.
