cren_camera_update

File: cren_camera.h

Type: C Function

Description

Updates a camera's position, orientation, and view matrix based on input states and time-based movement.

CREN_API void cren_camera_update(CRenCamera* camera, float timestep)

Parameters

camera:

Camera's memory address.

timestep:

Delta time value representing the time elapsed since the last frame update, measured in seconds.
