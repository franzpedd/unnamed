cren_camera_get_view_inverse

File: cren_camera.h

Type: C Function

Description

Gets the current camera inverse view matrix, this is faster than geting the view and inversing it's already internally handled.

CREN_API fmat4 cren_camera_get_view_inverse(CRenCamera* camera);

Parameters

camera:

Camera's memory address.

Return

Returns the last/current camera's inverse view matrix.
