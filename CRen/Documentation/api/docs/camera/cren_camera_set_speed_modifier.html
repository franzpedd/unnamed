cren_camera_set_speed_modifier

File: cren_camera.h

Type: C Function

Description

Sets/unsets the camera speed multiplier modifier.

CREN_API void cren_camera_set_speed_modifier(CRenCamera* camera, bool status, float value)

Parameters

camera:

Camera's memory address.

status:

true to enable apply speed modifier, false to disable

value:

The velocity modifier, should be more than 1.0f to be faster and less than 1.0f to be slower
