cren_camera_get_aspect_ratio

File: cren_camera.h

Type: C Function

Description

Gets the camera's aspect ratio.

CREN_API float cren_camera_get_aspect_ratio(CRenCamera* camera);

Parameters

camera:

Camera's memory address.

Return

Returns the last/current aspect ration uppon requesting, or 1.0f if camera's memory address is NULL.
