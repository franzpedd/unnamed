cren_camera_get_speed_modifier

File: cren_camera.h

Type: C Function

Description

Gets the camera 3D position in the world.

CREN_API bool cren_camera_get_speed_modifier(CRenCamera* camera, float* value)

Parameters

camera:

Camera's memory address.

value:

An optional address to retrieve the current modifier velocity.

Return

Returns the last/current camera's 3D position in the world.
