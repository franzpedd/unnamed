cren_camera_create

File: cren_camera.h

Type: C Function

Description

Creates and initializes a new camera instance with specified field of view and aspect ratio.

CREN_API CRenCamera* cren_camera_create(CRen_CameraType type, float initialAspectRatio, CRen_Renderer api);

Parameters

type:

What kind of camera is going to be created.

initialAspectRatio:

The division of viewport width and height, wich defines the aspect ratio

api:

Wich version of the Renderer API is used (only vulkan for now but if other API is supported the mathmatics changes)

Return

Returns either the camera memory address or NULL when the camera creation has failed, the console will contain information about the error if any has occured.
