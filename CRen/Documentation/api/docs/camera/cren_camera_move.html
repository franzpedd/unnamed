cren_camera_move

File: cren_camera.h

Type: C Function

Description

Starts the movement towards a direction without stop.

CREN_API void cren_camera_move(CRenCamera* camera, CRen_CameraDirection dir, bool moving);

Parameters

camera:

Camera's memory address.

dir:

Either one of the following: CREN_CAMERA_DIRECTION_FORWARD, CREN_CAMERA_DIRECTION_BACKWARD, CREN_CAMERA_DIRECTION_LEFT or CREN_CAMERA_DIRECTION_RIGHT

moving:

true to move, false to stop moving
