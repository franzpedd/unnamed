cren_quad_render

File: cren_primitives.h

Type: C Function

Description

Renders the quad, sending the draw command to the gpu.

CREN_API void cren_quad_render(CRenContext* context, CRenQuad* quad, CRen_RenderStage stage, const fmat4 modelMatrix);

Params

context:

CRen's context memory address.

quad:

The address of the opaque quad object.

stage:

The current render stage (picking or default), the callback to render will contain this info.

modelMatrix:

The quad model matrix, defining the quad's position, rotation and scale.
