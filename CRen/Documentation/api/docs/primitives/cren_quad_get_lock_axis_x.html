cren_quad_get_lock_axis_x

File: cren_primitives.h

Type: C Function

Description

Returns if the quad x axis is currently locked.

CREN_API bool cren_quad_get_lock_axis_x(CRenContext* context, CRenQuad* quad);

Params

context:

CRen's context memory address.

quad:

The address of the opaque quad object.

Return

Retruns true if x axis is locked, false otherwise.
