cren_quad_set_billboard

File: cren_primitives.h

Type: C Function

Description

Changes the behaviour of the quad.

CREN_API void cren_quad_set_billboard(CRenContext* context, CRenQuad* quad, bool value);

Params

context:

CRen's context memory address.

quad:

The address of the opaque quad object.

value:

Set it to true to make the quad act like a billboard, false to make the quad a normal object.
