cren_quad_set_lock_axis_y

File: cren_primitives.h

Type: C Function

Description

Changes the behaviour of the quad, lock the moving mechanism.

CREN_API void cren_quad_set_lock_axis_y(CRenContext* context, CRenQuad* quad, bool lock);

Params

context:

CRen's context memory address.

quad:

The address of the opaque quad object.

lock:

Set it to true if wish to lock the axis, false to unlock it.
