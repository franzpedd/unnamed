cren_texture2d_destroy

File: cren_primitives.h

Type: C Function

Description

Releases all resources used by a CRenTexture2D object.

CREN_API void cren_texture2d_destroy(CRenContext* context, CRenTexture2D* texture);

Params

context:

CRen's context memory address.

texture:

The texture's memory address.
