cren_quad_create

File: cren_primitives.h

Type: C Function

Description

Creates a CRenQuad object, a 2D plane that can be used for many pourpuses.

CREN_API CRenQuad* cren_quad_create(CRenContext* context, const char* albedoPath, uint32_t id);

Params

context:

CRen's context memory address.

albedoPath:

The path to the texture's albedo.

id:

Associates an identifier to the quad, using cren_create_id() is a good way to fill this.

Return

The address of the opaque object, used to track this particular resource.
