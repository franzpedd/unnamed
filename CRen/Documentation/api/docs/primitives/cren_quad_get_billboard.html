cren_quad_get_billboard

File: cren_primitives.h

Type: C Function

Description

Returns if the quad is currently acting as a billboard.

CREN_API bool cren_quad_get_billboard(CRenContext* context, CRenQuad* quad);

Params

context:

CRen's context memory address.

quad:

The address of the opaque quad object.

Return

Retruns true if acting as a billboard, false otherwise.
