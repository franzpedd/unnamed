cren_texture2d_create_from_buffer

File: cren_primitives.h

Type: C Function

Description

Creates a CRenTexture2D object, this is used when the image is sent via network, like Steam.

CREN_API CRenTexture2D* cren_texture2d_create_from_buffer(CRenContext* context, uint8_t* buffer, size_t bufferLen, int32_t width, int32_t height, bool uiTexture);

Params

context:

CRen's context memory address.

buffer:

The texture's binary data.

bufferLen:

The texture's binary data size in bytes.

width:

The texture's width.

height:

The texture's height.

uiTexture:

If the image is designed to be used on the UI set this true, false otherwise.

Return

The address of the opaque object, used to track this particular resource.

Expose: Currently the CRenTexture2D object is opaque without functions to expose it's members. Currently this is used internally only but it will change.
