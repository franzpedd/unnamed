cren_unregister_id

File: cren_context.h

Type: C Function

Description

Unregister an id into the tracking system, allowing it back to the pool of unused id's.

CREN_API bool cren_unregister_id(CRenContext* context, uint32_t id);

Parameters

context:

CRen's context memory address.

id:

The id to be registered.

Return

Returns true if successfully unregistered, false if the id was never registered or an error happened.
Warn: As of right now this function is not thread-safe, DON'T CALL IT FROM DIFFERENT THREADS.
