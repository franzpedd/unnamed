cren_pick_object

File: cren_context.h

Type: C Function

Description

CRen uses a dedicated render phase to write unique ids into objects that can be used for pixel perfect selection with this function.

CREN_API uint32_t cren_pick_object(CRenContext* context, float2 screenCoord);

Parameters

context:

CRen's context memory address.

screenCoord:

The pixel position on the screen to attempt picking.
