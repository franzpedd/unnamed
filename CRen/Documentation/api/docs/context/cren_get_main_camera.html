cren_get_main_camera

File: cren_context.h

Type: C Function

Description

The CRen initialization contains a main camera, this functions returns it.

CREN_API CRenCamera* cren_get_main_camera(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

The address of the main camera object, an opaque object itself used on camera functions.
