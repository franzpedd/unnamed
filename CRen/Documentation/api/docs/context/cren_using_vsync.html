cren_using_vsync

File: cren_context.h

Type: C Function

Description

Will return if CRen is currently using vertical-syncronization, matching the refresh-rate of the monitor (the one the window currently is).

CREN_API bool cren_using_vsync(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

true if vsync is enable, false otherwise.
