cren_set_render_callback

File: cren_context.h

Type: C Function

Description

This function assigns a callback to be executed during Vulkan's command buffer recording for rendering. It provides some control over the draw process, allowing you to decide which objects to render (since the drawing is a function per object) and, most importantly, the exact timing of when the drawing commands are issued within the frame's command sequence. 

CREN_API void cren_set_render_callback(CRenContext* context, CRenCallback_Render callback);

Parameters

context:

CRen's context memory address.

callback:

The callback function to be called.
