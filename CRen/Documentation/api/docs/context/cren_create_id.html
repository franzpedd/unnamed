cren_create_id

File: cren_context.h

Type: C Function

Description

CRen has a bounded, recyclable, memory efficient, O(1) for checking, ID allocator. Creates an unique id inside the renderer, that can assigned to an object on it's creation. 

CREN_API uint32_t cren_create_id(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

A unique id starting from 1.
Warn: As of right now this function is not thread-safe, DON'T CALL IT FROM DIFFERENT THREADS.
