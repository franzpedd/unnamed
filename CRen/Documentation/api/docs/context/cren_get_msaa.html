cren_get_msaa

File: cren_context.h

Type: C Function

Description

CRen render has support for MSAA (Multisample Anti-Aliasing), this will return how many samples is currently set.

CREN_API CRen_MSAA cren_get_msaa(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

Either of the following: CREN_MSAA_X1 (Turned-Off), CREN_MSAA_X2, CREN_MSAA_X4, CREN_MSAA_X8, CREN_MSAA_X16, CREN_MSAA_X32 or CREN_MSAA_X64.
