cren_are_validations_enabled

File: cren_context.h

Type: C Function

Description

If the CRen initialization hint for validations were set to true this will return true, false otherwise.

CREN_API bool cren_are_validations_enabled(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

true if validations are enabled, false if not.
