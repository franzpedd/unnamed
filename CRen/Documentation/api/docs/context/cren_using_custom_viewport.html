cren_using_custom_viewport

File: cren_context.h

Type: C Function

Description

Will return if CRen is currently using the viewport renderphase underneath it for rendering the game in a sub-window.

CREN_API bool cren_using_custom_viewport(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

true if using a custom viewport, false otherwise.
