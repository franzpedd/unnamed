cren_get_vulkan_backend

File: cren_context.h

Type: C Function

Description

Allows the access to Vulkan objects for directly controll of the renderer.

CREN_API void* cren_get_vulkan_backend(CRenContext* context);

Parameters

context:

CRen's context memory address.

Return

An opaque memory address that can be casted to CRenVulkanBackend* in order to access it's members. (Will be changed into an opaque object in the future).
