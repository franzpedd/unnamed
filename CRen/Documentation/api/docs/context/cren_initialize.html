cren_initialize

File: cren_context.h

Type: C Function

Description

Initializes the CRen library.

CREN_API CRenContext* cren_initialize(const CRenCreateInfo createInfo);

Parameters

createInfo:

Struct containing all the hints for the library initialization.

Return

Returns the CRenContext, an opaque object used for various functions, NULL if the creation has failed. The terminal will contain more info if it has failed.
