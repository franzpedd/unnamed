cren_set_create_vulkan_surface_callback

File: cren_context.h

Type: C Function

Description

This function registers a callback for surface creation. Since CRen does not manage the window itself, this must be implemented by the client's chosen windowing system (e.g., GLFW, SDL, or a platform-specific API). 

CREN_API void cren_set_create_vulkan_surface_callback(CRenContext* context, CRenCallback_CreateVulkanSurfaceCallback callback);

Parameters

context:

CRen's context memory address.

callback:

The callback function to be called.
