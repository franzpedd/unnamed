cren_set_ui_image_count_callback

File: cren_context.h

Type: C Function

Description

This function assigns a callback to be executed after the swapchain image count has changed, this is mostly(only) used for UI and exists to provide usability with Dear ImGui. 

CREN_API void cren_set_ui_image_count_callback(CRenContext* context, CRenCallback_ImageCount callback);

Parameters

context:

CRen's context memory address.

callback:

The callback function to be called.
