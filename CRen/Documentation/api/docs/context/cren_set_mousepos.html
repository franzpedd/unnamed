cren_set_mousepos

File: cren_context.h

Type: C Function

Description

Informs the renderer where the cursor is, this is used internally for picking renderphase optimization(the only pixel rendered is underneath the cursor position).

CREN_API void cren_set_mousepos(CRenContext* context, const float2 pos);

Parameters

context:

CRen's context memory address.

pos:

The cursor's current position.
