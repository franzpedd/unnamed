cren_set_resize_callback

File: cren_context.h

Type: C Function

Description

This function assigns a callback to be executed once the renderer has applyed the resizing event, (sent via functions: cren_resize/cren_minimize/cren_restore/cren_maximize/), Usefull when user wants to further resize something after the renderer was resized, but it's not a necessary callback to be set. 

CREN_API void cren_set_resize_callback(CRenContext* context, CRenCallback_Resize callback);

Parameters

context:

CRen's context memory address.

callback:

The callback function to be called.
