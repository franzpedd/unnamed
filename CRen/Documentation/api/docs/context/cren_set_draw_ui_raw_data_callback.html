cren_set_draw_ui_raw_data_callback

File: cren_context.h

Type: C Function

Description

This function assigns a callback to be executed when it's time to draw UI objects (usually after the world objects or the UI would be behind it). Most user interfaces bundles all of it's "widgets" together into a "raw" data, this function works like render for objects. 

CREN_API void cren_set_draw_ui_raw_data_callback(CRenContext* context, CRenCallback_DrawUIRawData callback);

Parameters

context:

CRen's context memory address.

callback:

The callback function to be called.
