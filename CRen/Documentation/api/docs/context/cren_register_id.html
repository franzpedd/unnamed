cren_register_id

File: cren_context.h

Type: C Function

Description

Register an id into the tracking system.

CREN_API bool cren_register_id(CRenContext* context, uint32_t id);

Parameters

context:

CRen's context memory address.

id:

The id to be registered.

Return

Returns true if successfully registered, false if either the context was not initialized or the id is already registered.
Warn: As of right now this function is not thread-safe, DON'T CALL IT FROM DIFFERENT THREADS.
