cren_set_user_pointer

File: cren_context.h

Type: C Function

Description

This function attachs an arbitrary user-owned data to a library-owned object and allows latter access to the pointer using the get function. 

CREN_API void cren_set_user_pointer(CRenContext* context, void* pointer);

Parameters

context:

CRen's context memory address.

pos:

The cursor's current position.
