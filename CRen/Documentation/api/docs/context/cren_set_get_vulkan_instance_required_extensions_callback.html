cren_set_get_vulkan_instance_required_extensions_callback

File: cren_context.h

Type: C Function

Description

This callback is required for Vulkan instance creation to query the necessary window system extensions (e.g., VK_KHR_xlib_surface or VK_KHR_wayland_surface). While the required extensions can be auto-detected on Windows and Android, they cannot be reliably determined on Linux due to multiple possible display servers (X11, Wayland). This platform-specific limitation necessitates the callback, though it may be removed in the future if a reliable auto-detection method becomes available for Linux. 

CREN_API void cren_set_get_vulkan_instance_required_extensions_callback(CRenContext* context, CRenCallback_GetVulkanRequiredInstanceExtensions callback);

Parameters

context:

CRen's context memory address.

callback:

The callback function to be called.
