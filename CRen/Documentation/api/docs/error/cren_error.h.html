cren_error.h

Overview

Defines helper functions and macros for error-tracking.

Code implementation

Code implementation can be found here

Macro

#ifdef CREN_LOG_DISABLED
    #define CREN_LOG(...)
    #define CREN_ASSERT(condition, msg) ((void)0)
#else
    #define CREN_LOG(severity, ...) cren_log_message(severity, __FILE__, __LINE__, __VA_ARGS__);
    #define CREN_ASSERT(condition, ...) if (!(condition)) { cren_log_message(CREN_LOG_SEVERITY_FATAL, __FILE__, __LINE__, __VA_ARGS__); }
#endif

If CREN_LOG_DISABLED is defined, errors will be skipped but applications will be faster.

Parameters

condition:

Given a boolean condition, if that's false the assertion will be trown. (ASSERT ONLY)

msg:

A text string message about the assertion. (ASSERT ONLY)

severity:

The severity of the log message. (LOG ONLY)

...:

The formated string message about the log message. (LOG ONLY)
