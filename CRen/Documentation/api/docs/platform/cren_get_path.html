cren_get_path

File: cren_error.h

Type: C Function

Description

Utility to retrieve the asset's correct path, since paths may differ from platform to platform.

CREN_API void cren_get_path(const char* subpath, const char* assetsRoot, bool removeExtension, char* output, size_t outputSize);

Usage

const char* root = "../data";
char defaultVert[CREN_PATH_MAX_SIZE];
cren_get_path("shaders/compiled/quad.vert.spv", root, 0, defaultVert, sizeof(defaultVert));
Then defaultVert will correctly have the quad.vert.spv file.
