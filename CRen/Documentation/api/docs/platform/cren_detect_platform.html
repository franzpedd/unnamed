cren_detect_platform

File: cren_error.h

Type: C Function

Description

Returns what platform the system is currently on.

CREN_API CRen_Platform cren_detect_platform();

Return

One of the following: CREN_PLATFORM_UNKNOWN_APPLE, CREN_PLATFORM_MACOS, CREN_PLATFORM_IOS, CREN_PLATFORM_X11, CREN_PLATFORM_LINUX, CREN_PLATFORM_ANDROID, CREN_PLATFORM_WINDOWS or CREN_PLATFORM_UNKNOWN.
