cren_load_file

File: cren_error.h

Type: C Function

Description

Utility to load a file from device, usually used with cren_get_path.

CREN_API unsigned int* cren_load_file(const char* path, size_t* outSize);

Usage

const char* root = "../data";
char defaultVert[CREN_PATH_MAX_SIZE];
cren_get_path("shaders/compiled/quad.vert.spv", root, 0, defaultVert, sizeof(defaultVert));
size_t spirvSize = 0;
unsigned int* spirvCode = cren_load_file(defaultVert, &spirvSize);

Params

path:

The disk/device path of the asset.

outSize:

Address of a size_t object to hold asset's size in bytes after loaded.
