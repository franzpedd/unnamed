cren_stbimage_load_from_file

File: cren_error.h

Type: C Function

Description

This function is a wrapper of stb_load so it's not needed to expose stb outside the library.

CREN_API uint8_t* cren_stbimage_load_from_file(const char* path, int desiredChannels, int* outWidth, int* outHeight, int* outChannels);

Parameters

path:

Image's path on disk.

desiredChannels:

How many channels to read from the image. 1 = R, 2 = RG, 3 = RGB, 4 = RGBA.

outWidth:

Address to an integer to store image's width once loaded.

outHeight:

Address to an integer to store image's height once loaded.

outChannels:

Address to an integer to store image's channels amount once loaded.

Return

The binary (char*/uint8_t*) that represents such image.
