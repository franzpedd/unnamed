align_as

File: cren_defines.h

Type: C Macro

Description

The preprocessor checks which compiler is being used:
- Checks if compiling as C++11 or newer;
- Checks if it's Microsoft Visual C++;
- Checks if it's GCC or Clang;
- If none of the above, falls back to the C11 standard; 

#if defined(__cplusplus) && __cplusplus >= 201103L
    #define align_as(X) alignas(X)  // C++11 native
#elif defined(_MSC_VER)
    #define align_as(X) __declspec(align(X))
#elif defined(__GNUC__) || defined(__clang__)
    #define align_as(X) __attribute__((aligned(X)))
#else
    #define align_as(X) _Alignas(X)  // C11 native
#endif

Parameters

X:

Integer to align the object as, multiple of 2.
