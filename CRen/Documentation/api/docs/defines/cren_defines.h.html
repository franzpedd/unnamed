Definitions

Concurrently rendered frames:

#define CREN_CONCURRENTLY_RENDERED_FRAMES 2

Define the number of frames that can be in-flight simultaneously, how many frames worth of resources is maintain concurrently to prevent CPU and GPU from waiting on each other.
- Prevents Stalls: Without pipelining, CPU must wait for GPU to finish before starting next frame;
- Hides Latency: GPU can work on frame N while CPU prepares frame N+1;

Path maximum size

#define CREN_PATH_MAX_SIZE 128

Paths to data files like textures, fonts and any resource must be at most this size, you may overwrite this if necessary.

Compile-time array size calculator

#define CREN_STATIC_ARRAY_SIZE(ARR) ((int)(sizeof(ARR) / sizeof(*(ARR))))

Calculates the number of elements in a static (fixed-size) array. (Does NOT work with pointers).
